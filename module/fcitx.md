# `fcitx`

`fcitx` is the preferred Chinese input method.

1. Install relevant packages: `sudo pacman -Syu fcitx5 fcitx5-rime librime fcitx5-configtool`.
2. Set `Fcitx 5` as the Virtual Keyboard.
3. Configure `Input Method` to use Rime.
4. Add-Ons -> Unicode -> Trigger Key (disable).
	1. Possibly make an exception for Unicode inputs from `typography` `Ctrl+Shift+U`.
5. On-screen keyboard must be activated (via taskbar icon which is usually hidden) (and set to Fcitx5 in virtual keyboard settings).
